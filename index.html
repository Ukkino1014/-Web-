きっかけは技術の授業から？

正直中学で技術の授業やって軽くウェブサイトの作り方を学ぶまではコードを書く系のプログラミングとかほとんどやったことなくて、あるとすれば親父に教育されてちょっとだけPythonをやったくらい。（Pythonも後々書いてみたいなぁ）

んで、とにかく、２年生の途中くらいから、技術の授業で、「ライフイズテック レッスン」っていう、中学生向けのHTML・CSSを学ぶソフト？というかウェブからアクセスできる練習サイトをやり始めたわけですよ。まあ正直、基本的にはそのサイトのキャラクターがしゃべってくれるコードをひたすら書き写しまくって、形上ウェブサイトを作ったみたいな達成感を味わえるもので、最初の方は、というか今現在授業で学んだところは独創性皆無なんだけど、もうすぐ自分たちでウェブサイトを作るっていうのをやるかもしれないかもしれないかもしれないらしくて、ちょっと楽しみかもしれないかもしれないかもしれない。

Microsoft VS Code が便利すぎぃ！

今この文面も、Microsoft VS(Visual Studio) Code を使って書いてるんですけど、なかなかに結構便利。ウェブサイト作り初心者な僕でもミスなくウェブサイトが作れるように、文字の色分けとか、スペルチェックとか、ライブでコードの修正をウェブサイトに適応してくれるとか、たくさん便利な拡張機能が用意されてます！そのおかげでなかなかにスムーズにこのサイトを作れている。
